# 🏢 Real Estate Sales & Performance Dashboard (Power BI)

## 📌 Overview

This project presents an end-to-end data analytics solution using Power BI to analyze real estate business performance. It transforms raw data into meaningful insights, helping stakeholders understand sales trends, customer behavior, agent performance, and overall business efficiency.

---

## 🎯 Goals and Objectives

### 🔹 Goals

* To build a complete data analytics solution using Power BI
* To convert raw real estate data into actionable insights
* To design an efficient and scalable star schema data model

### 🔹 Objectives

* To analyze sales performance across cities and time periods
* To evaluate agent performance and lead conversion rates
* To understand customer behavior through segmentation
* To track financial metrics such as profit, expenses, and rent
* To create interactive dashboards for data-driven decision-making

---

## 🧱 Data Model

The project is designed using a **Star Schema Data Model**, which ensures efficient querying and analysis.

### 🔸 Fact Tables

* Fact_Sales
* Fact_Leasing
* Fact_Leads
* Fact_Expenses

### 🔸 Dimension Tables

* Dim_Date
* Dim_Property
* Dim_Customer
* Dim_Agent
* Dim_Location

---

## 🔄 Workflow

1. Data Collection (Excel dataset)
2. Data Cleaning (Power Query)
3. Data Transformation
4. Data Modeling (Star Schema)
5. DAX Calculations
6. Dashboard Development
7. Testing and Validation

## 📈 Key Insights

* Sales remained stable across years, with peak performance in 2022
* Expert agents contributed the highest sales
* Buyers generated more revenue compared to tenants
* Customers aged 30–40 formed the most active segment
* Delhi emerged as the top-performing city
